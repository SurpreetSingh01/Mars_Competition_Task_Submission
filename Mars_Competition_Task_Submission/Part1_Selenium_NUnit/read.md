# Part 1 – Selenium Automation (C# + NUnit)

## Overview
This part contains manual test cases and Selenium automation for the Mars Competition Task.

The automation framework is implemented using:
- Selenium WebDriver
- C#
- NUnit
- Page Object Model (POM)

## Project Structure

Part1_Selenium_NUnit
- SourceCode – Selenium automation project
- TestCases – Manual test case Excel file
- TestData – Test data files used in automation
- Reports – Automation execution reports
- Screenshots – Evidence of test execution

## Tools Used
- Selenium WebDriver
- C#
- NUnit
- Visual Studio
- GitHub

## Features Implemented
- Login automation
- Profile related test scenarios
- Data-driven test execution
- Page Object Model framework

## How to Run
1. Open the solution in Visual Studio
2. Restore NuGet packages
3. Run tests using NUnit Test Explorer

## Notes
Build folders such as `bin` and `obj` are excluded from the repository.
